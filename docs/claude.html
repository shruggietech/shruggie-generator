SHRUGGIE GENERATOR

An emotion set based on the Affective Dynamics Framework by ShruggieTech

▼Emotion

Select a mood mapped to the Affective Dynamics Framework. Positive, negative, and complex emotional states each produce unique facial expressions and ambient effects.

▼Face

Fine-tune the expression. Eye and mouth sliders modify the selected mood's base look with subtle variations in shape and positioning.

Ghost mode (empty head) ?Removes fills and draws the entire character as a monochrome outline using the selected color. Not changed by Randomize.
▼Color

Set the character's primary color. Body, skin tone, accessories, and leg color are all derived from this selection.

▼Background

Controls the preview backdrop and the downloaded SVG. Transparent mode exports without a background fill, suitable for compositing.

▼Pose

Adjust arm positions independently or lock them together with the mirror toggle. The shrug intensity controls how high the arms raise.

▼Accessories

Equip cosmetic items. When rules are enabled, conflicting combinations (such as Top Hat with Good disposition's halo) are automatically resolved.

▼Rules

Safety rails that prevent hard-to-read or messy outputs. All rules are active by default. Disable individual rules for full creative freedom.
